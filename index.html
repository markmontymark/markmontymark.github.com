Getting Jenkins Git plugin to resolve 'Auth fail' (ie handle SSH Keys)

Turns out JGit plugin doesn't handle .ssh/ keys as well as commandline git.A The fix 
nohup java -Dorg.jenkinsci.plugins.gitclient.Git.useCLI=true -jar jenkins.war > jenkins.log 2>&1 &
The source 
https://issues.jenkins-ci.org/browse/JENKINS-17204

For those of you who are used to starting Jenkins from terminal, here's the command that made it work for me (with the important bit starting after the java and before the -jar):
nohup java -Dorg.jenkinsci.plugins.gitclient.Git.useCLI=true -jar ~/Documents/jenkins.war --httpPort=8081 --ajp13Port=8010 > /tmp/jenkins.log 2>&1 &
